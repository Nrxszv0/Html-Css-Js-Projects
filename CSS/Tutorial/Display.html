Display

The display property specifies how an element is displayed. Each element has a default display value depending on the element type. The default display value for most is block or inline.

Block Level

They take up the full available width

Inline

They don't start on a new line and only take up necessary space.

Display:none;

JavaScript uses display:none; to hide and show elements without deleting and recreating them.

Override the Default Display Value

Changing an inline element to a block element or vice versa can make a page look a specific way.
