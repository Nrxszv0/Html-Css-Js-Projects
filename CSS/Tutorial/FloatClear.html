Float and Clear

Float

the float property is used for positioning and formatting. It can have the values of

  • left- elements float to left of its container
  • right- elements float to right of its container
  • none- element does not float. This is default
  • inherit- element inherits the float value of its parent

Clear

The clear property specifies what elements can float beside the cleared element and on which side. It can have the values of

  • none- Floating elements on bot sides. Default
  • left- no floating elements on left side
  • right- no floating elements on right side
  • both- no floating elements on either side
  • inherit- element inherits value of parent

Without Clear

d1
d2- D2 is after d1. Since d1 floats to the left the text in d2 flows around d1.


With Clear

d3
d4- clear:left; moves div4 down below the floating d3. The value of left clears elements floated to the left. You could also clear right and both.

Clearfix hack

If an element is taller than the element containing it and is floated it will overflow outside of its container.
