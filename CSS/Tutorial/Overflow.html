Overflow

The overflow property specifies whether to clip the content or add scrollbars when the content of an element is too big. The property has the values of:

  • visible- the default value, the overflow is not clipped, it renders outside the element's box.
  • hidden- the overflow is clipped and the rest of the content is invisible
  • scroll- the overflow is clipped and a scrollbar is added.
  • auto- similar to scroll but only adds a scrollbar when necessary.

Overflow X and Y

The overflow-x and overflow-y properties specify whether to change the overflow of content horizontally, vertically, or both. overflow-x specifies what to do with the left/right edges. overflow-y specifies what to do with the top and bottom edges.
