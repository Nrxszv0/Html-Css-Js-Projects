Links

Links can be styled with any property. They can also be styled depending on their state. The four states are

  • a:link- a normal unvisited link
  • a:visited- a link that has been visited by the user
  • a:hover- a link when the user hovers over it with the mouse.
  • a:active- a link the moment it is clicked.
