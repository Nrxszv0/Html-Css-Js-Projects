Backgrounds

There are background properties used to define the background effects for elements.

Summary

  • background- Sets all the background properties in one declaration.
  • background-attachment- Sets whether a background image is fixed to a certain part of a page or it scrolls as the user scrolls
  • background-clip- Specifies the background's painting area
  • background-color- Sets the element's background color
  • background-image- Sets the element's background image
  • background-origin- Specifies where the background image is positioned
  • background-position- Sets the starting position of a background image
  • background-repeat- Sets how a background image will be repeated
  • background-size- Specifies the size of the background image
