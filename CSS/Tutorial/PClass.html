Pseudo-Classes

A pseudo class defines a special state of an element. It can be used to

  • Style an element when a user mouses over it
  • Style visited and unvisited links differently
  • Style an element when it gets focus

The :first-child Pseudo-class

The :first-child matches a specified element that is the first child of another element.
