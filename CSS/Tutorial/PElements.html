Pseudo-Elements

A pseudo element is used to style specified parts of an element. They use a double colon to distinguish from single colon pseudo classes. It can be used to

  • Style the first letter or line of an element.
  • Insert content before or after an element's content.

::first-line

The properties it applies to are

  • font
  • color
  • background
  • word-spacing
  • letter-spacing
  • text-decoration
  • vertical-align
  • text-transform
  • line-height
  • clear
